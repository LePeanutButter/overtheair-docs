## Background

Este repositorio contiene el trabajo académico enfocado en el análisis de seguridad de imágenes de contenedores como componente fundamental del pipeline DevSecOps. La investigación aborda la implementación de controles automáticos de escaneo en entornos cloud-native, con especial atención en metodologías de detección de vulnerabilidades aplicadas a capas de contenedor y su integración en flujos de entrega continua.

El estudio propone un enfoque integral que combina automatización, priorización contextual de riesgos y validación continua dentro de una arquitectura DevSecOps resiliente, abordando los principales retos técnicos y las mejores estrategias para maximizar la efectividad del escaneo de contenedores.

## Documentation

### Temas Cubiertos

La investigación cubre los siguientes temas fundamentales:

- **DevSecOps**: Integración de seguridad en el ciclo de vida del desarrollo
- **Container Image Scanning**: Análisis de vulnerabilidades en imágenes de contenedor
- **SBOM (Software Bill of Materials)**: Generación y análisis de inventarios de componentes
- **CI/CD Security**: Controles de seguridad en pipelines de integración continua
- **Kubernetes Security**: Seguridad en orquestación de contenedores
- **Vulnerability Management**: Gestión y priorización de vulnerabilidades

### Metodología de Investigación

El trabajo académico se basa en:

- **Análisis teórico**: Revisión de estándares y mejores prácticas (NIST SSDF, OWASP, ISO/IEC 27001)
- **Evaluación de herramientas**: Análisis comparativo de herramientas de escaneo industrial
- **Estudios de caso**: Implementación práctica en entornos reales
- **Métricas de efectividad**: Evaluación cuantitativa de controles de seguridad

### Referencias Académicas

La investigación se fundamenta en estándares y marcos de trabajo reconocidos:

- **NIST SSDF**: Secure Software Development Framework
- **OWASP**: Open Web Application Security Project
- **MITRE ATT&CK**: Framework de tácticas y técnicas de adversarios
- **SLSA**: Supply-chain Levels for Software Artifacts
